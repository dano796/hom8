# HomeFlow

Android app for home task and expense management, built with Kotlin MVVM, Room, Firebase, and Hilt.

## Tech Stack

- Kotlin + Coroutines
- MVVM + LiveData / StateFlow
- Room Database (local-first)
- Firebase Auth + Firestore + Storage
- Hilt (dependency injection)
- Navigation Component

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/dano796/home-flow.git
```

### 2. Configure Firebase

This project requires a `google-services.json` file that is **not included in the repository** for security reasons.

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Create a project (or use an existing one)
3. Register an Android app with package name `com.homeflow.app`
4. Download `google-services.json`
5. Place it in the `app/` folder:

```
app/
└── google-services.json   ← place it here
```

6. In Firebase Console, enable:
   - **Authentication** → Email/Password
   - **Firestore Database**
   - **Storage**

### 3. Build and run

Open the project in Android Studio and run on a device or emulator (min SDK 24).

## Files NOT included in this repo

| File                       | Reason                                                                  |
| -------------------------- | ----------------------------------------------------------------------- |
| `app/google-services.json` | Contains Firebase API keys — generate your own from Firebase Console    |
| `local.properties`         | Contains your local Android SDK path — auto-generated by Android Studio |
| `*.keystore` / `*.jks`     | App signing keys — never share these                                    |
| `keystore.properties`      | Signing configuration with passwords                                    |
| `build/`                   | Compiled output                                                         |
